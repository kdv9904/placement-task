# Installation Guide

## Prerequisites
- Node.js (version 18 or higher)
- Git
- npm or yarn

## Installation Steps

### 1. Clone Repository
git clone https://github.com/kdv9904/placement-task.git
cd placement-task


### 2. Create uploads folder in root directory
mkdir -p uploads

### 3. Run the Application
npm run dev

### 4. Install Additional Libraries if needed
npm i pdf-parse
npm i mammoth
