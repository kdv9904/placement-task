# Installation Guide

## Prerequisites
- Node.js (version 18 or higher)
- Git
- npm or yarn

## Installation Steps

### 1. Clone Repository
git clone <repository-url>
cd prompt-refinement-system


### 2. Create uploads folder in root directory
mkdir -p uploads

### 3. Run the Application
npm run dev

### 4. Install Additional Libraries if needed
npm i pdf-parse
npm i mammoth
